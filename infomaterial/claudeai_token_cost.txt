 Claude Pro Subscription

  Claude Pro ($20/month) provides:
  - ~5x more usage than free tier
  - No specific published token limit (it's usage-based with higher caps)
  - Priority access during high traffic
  - Access to Claude Opus 4.5, Sonnet, and Haiku

  Usage limits are dynamic and not publicly disclosed as exact token counts.

  To Check Your Usage

  1. Claude Pro (claude.ai): Go to Settings → Usage in your Claude account
  2. API usage: Visit https://console.anthropic.com → Usage

  Claude Code Context

  You're using Claude Opus 4.5 through Claude Code. If you're hitting limits, you can:
  - Use /compact to summarize conversation history
  - Use --model haiku or --model sonnet for simpler tasks
  - Check usage at console.anthropic.com if using API key
